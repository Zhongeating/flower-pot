## 为什么总方案放在 00

总方案不是单一“产品定义”文件，而是整合了：

- 产品定位
- 用户研究结论
- 市场与竞品判断
- 技术路线取舍
- 商业与验证计划
- 汇报表达结构

因此它应作为全项目入口，而不是塞进 01_产品定义。01 更适合存放产品定义专题、预研和子版本定义。
