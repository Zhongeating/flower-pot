# Flower Pot Project

This repository contains the product proposal, research evidence, technical plan, CAD model files, and presentation assets for the smart planter project.

## Structure

- `00_项目总览`: current proposal, HTML presentation, and version notes.
- `01_产品定义`: product definition entry and positioning notes.
- `02_研究报告`: market, competitor, and user evidence.
- `03_用户研究`: user validation plan.
- `04_技术方案`: irrigation decisions, system design, software PRD, and CAD model files.
- `05_执行文档`: execution document entry.
- `06_素材资源`: visual assets used by the proposal and presentation.

## How To Review

Start with the current proposal and HTML presentation under `00_项目总览`, then review the system decision report and CAD files under `04_技术方案`.
